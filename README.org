#+title: consult-symbol
#+author: Daniel Fleischer

Search all Emacs symbols with [[https://github.com/minad/consult][consult]]-powered narrowing by category.

[[file:assets/image_2.png]]

* Overview

=consult-symbol= provides a single interactive command that lets you search
across all Emacs symbols — commands, functions, macros, variables, faces,
custom groups, and more — with category narrowing, rich annotations, and
history-based sorting.


[[file:assets/image_1.png]]



* Features

- *Category narrowing* — press the narrowing key to filter by type:
  | Key | Category        |
  |-----+-----------------|
  | =c=   | Command         |
  | =f=   | Function        |
  | =m=   | Macro           |
  | =F=   | Special Form    |
  | =v=   | Variable        |
  | =u=   | Custom Variable |
  | =a=   | Face            |
  | =G=   | Custom Group    |
  | =t=   | CL Type         |
- *Rich annotations* — symbol class, current value (for variables), face
  preview, and first-line docstring
- *History sorting* — recently selected symbols appear first
- *Embark integration* — automatic type detection for embark actions
- *Customizable* — action function, annotation widths, narrowing categories

* Requirements

- Emacs 27.1+
- [[https://github.com/minad/consult][consult]]
- [[https://github.com/minad/marginalia][marginalia]]

Optional:
- [[https://github.com/Wilfred/helpful][helpful]] — richer symbol descriptions
- [[https://github.com/oantolin/embark][embark]] — contextual actions on symbols

* Installation

** With use-package and package-vc (Emacs 29+)

#+begin_src emacs-lisp
(use-package consult-symbol
  :vc (:url "https://github.com/danielfleischer/consult-symbol"
       :rev :newest)
  :bind ("C-h o" . consult-symbol))
#+end_src

** With elpaca/straight

#+begin_src emacs-lisp
(use-package consult-symbol
  :elpaca (:host github :repo "danielfleischer/consult-symbol")
  :bind ("C-h o" . consult-symbol))
#+end_src

** Manual

Clone this repository and add it to your =load-path=:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/consult-symbol")
(require 'consult-symbol)
(keymap-global-set "C-h o" #'consult-symbol)
#+end_src

* Embark Integration

=consult-symbol= provides built-in support for [[https://github.com/oantolin/embark][embark]], enabling contextual actions on symbols based on their type.

** Setup

To enable embark integration, add this to your configuration:

#+begin_src emacs-lisp
(use-package consult-symbol
  :config
  (consult-symbol-setup-embark))
#+end_src

Or if you're not using =use-package=:

#+begin_src emacs-lisp
(require 'consult-symbol)
(consult-symbol-setup-embark)
#+end_src

* Customization

All options are under the =consult-symbol= customize group.

- =consult-symbol-action= — function called with the selected symbol (default:
  describe via =helpful-symbol= or =describe-symbol=)
- =consult-symbol-history= — history variable to use for symbol search
   + =nil= — use default minibuffer history
   + ='consult-symbol--history= — use dedicated history (default)
   + Any symbol — use custom history variable
- =consult-symbol-types= — alist of narrowing keys and category labels
- =consult-symbol-include-internal= — whether to include internal symbols
  (those containing =--=)
- =consult-symbol-doc-width= — max width for docstrings in annotations
- =consult-symbol-value-width= — max width for variable values in annotations
- =consult-symbol-face-sample= — sample text for face previews
